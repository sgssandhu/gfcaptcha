Implement and have full control over a captcha field for your Gravity Forms

== Description ==

Provides an alternative to the reCaptcha field built into Gravity Forms, by integrating the Securimage PHP CAPTCHA system as a field for your forms.

Full feature control integrated into Gravity Forms Settings page.

= Features =
* *Customizable code length, character sets, and Unicode support*
* *TTF font support*
* *Easily add background images*
* *Several security features such as image distortion, random lines, and noise*
* *Flash button to stream audible codes in WAV format*
* *Ability to use a word list*
* *Case sensitive option for added security*
* *Display alphanumeric captchas, or simple math problems*
* *Highly customizable!*

[PHP Captcha]: http://www.phpcaptcha.org/ "free PHP CAPTCHA script"
[Gravity Forms]: http://www.gravityforms.com/ "Gravity Forms"

== Installation ==

1. Unzip, upload plugin folder to your plugins directory (`/wp-content/plugins/`)
2. Activate plugin
3. Configure Captcha settings from the Gravity Forms Settings page.
4. Use the `CustomCaptcha` button in the `Advanced Fields`
